# Contributing to Dew Point Calculator

Thank you for your interest in contributing! Here's how you can help.

## Reporting Bugs

If you find a bug:

1. Check if it's already reported in [Issues](https://github.com/iret33/ha-dew-point-calculator/issues)
2. If not, create a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Home Assistant version
   - Integration version
   - Relevant logs (with sensitive data removed)

## Feature Requests

Have an idea? Open an issue with:
- Clear description of the feature
- Use cases
- Why it would be valuable

## Pull Requests

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Test thoroughly
5. Commit with clear messages
6. Push and create a Pull Request

### Code Style

- Follow Home Assistant's [development guidelines](https://developers.home-assistant.io/docs/development_guidelines)
- Use type hints
- Add docstrings to functions
- Keep code clean and readable

## Testing

Before submitting:

1. Test the integration manually
2. Verify it works with different sensor types
3. Check logs for errors
4. Ensure config flow works correctly

## Questions?

Open an issue or start a discussion!
